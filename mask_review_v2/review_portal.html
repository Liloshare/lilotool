검수 작업을 단계별로 진입하세요

번들/폴리곤 검수부터 납품까지 작업 흐름을 한 화면에서 확인하고, 필요한 검수 페이지와 관련 자료로 바로 이동합니다. 각 검수 페이지는 로컬 파일을 불러와 브라우저에서 확인하고 결과 파일을 내려받는 방식으로 사용합니다.

관련 자료

검수 단계

왼쪽에서 오른쪽 순서로 진행

1

번들/폴리곤 검수

CVAT에서 번들 단위 작업 상태와 기본 라벨링 결과를 먼저 확인합니다.

  • 번들 작업 확인
  • 라벨링 상태 확인
  • 후속 검수 대상 파악
CVAT 열기
2

바코드 매칭

아이템별 바코드 매칭 상태와 오류 항목을 검수하는 페이지입니다.

  • 아이템별 오류 확인
  • 오류 유형 기록
  • 결과 엑셀 내보내기
검수 시작
3

주문목록 불일치 csv 추출

주문 조회 자료와 검수 기준을 비교해 주문목록 불일치 확인용 CSV를 준비합니다.

  • 주문목록 비교
  • 불일치 후보 추출
  • 검수 입력 파일 준비
CSV 추출
4

주문번호 불일치 검수

정답지 엑셀과 주문번호별 annotation을 비교해 불일치 항목을 검수하는 페이지입니다.

  • 정답지 엑셀 불러오기
  • 주문번호별 불일치 확인
  • JSON 수정본 및 결과 엑셀 저장
검수 시작
5

폴리곤 재작업

검수에서 발견된 폴리곤 오류, 누락, 삭제, 그룹 처리 항목을 기준으로 재작업합니다.

  • 오류 항목 확인
  • JSON 수정본 반영
  • 재검수 대상 정리
CVAT 열기
6

납품

최종 검수 결과와 수정 파일을 확인한 뒤 납품용 산출물을 정리합니다.

  • 최종 결과 확인
  • 파일 누락 점검
  • 납품 자료 정리
